# Sales Pitch Deck: Aldvra

## Deck Design Rules

- Format: 16:9 presentation slide image.
- Style: McKinsey-style sales pitch deck, clean white background, sharp grid, muted navy/charcoal text, one teal accent.
- Composition: about 80% visual exhibit, 10% text, 10% whitespace and source line.
- Text: one action title plus very short labels only. Avoid paragraphs inside the image.
- Each slide must sell one message.
- Use simple vector-style business graphics, not decorative illustrations.
- Do not use real company logos.
- Do not claim guaranteed profit or investment advice.

## Sales Narrative

0. Aldvra is introduced as the controlled intelligence layer for modern trading workflows.
1. Trading customers want guided action, not more alerts.
2. Businesses need AI adoption without unsafe execution risk.
3. The platform turns broker-connected trading into a supervised AI workflow.
4. The current product already covers the core capability base.
5. The same platform can power B2C and B2B revenue paths.
6. The ask is to turn the current base into a commercial-grade product.
7. The investment case is measurable through four TCO buckets.

**Speaker note:**

Use this slide to make the investment ask concrete. The TCO model is built from four buckets. First is development: 4 backend developers, 2 frontend developers, 2 QA engineers, 2 DevOps engineers, 1 UX designer, and 1 lead engineer or technical architect. Second is cloud infrastructure, which should be estimated using the selected provider calculator for AWS, GCP, or Azure. Third is LLM cost, which depends on model choice, token volume, and agent run frequency. Fourth is the broker platform cost, including Kite Connect API and market data access. The right sales message is that TCO is not a black box; it can be controlled once usage assumptions are clear.
